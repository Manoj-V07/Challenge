About Gemini Code Helper

This extension uses Chrome's built-in AI (Gemini Nano) to bring you powerful code assistance without your data ever leaving your device.

To use this extension, you need:

  • Google Chrome version 127 or newer.
  • The built-in AI model enabled. You can enable it by navigating to chrome://settings/ai and turning on the feature.

How to Use

  1. Find a code snippet on any website.
  2. Highlight the code with your mouse.
  3. Right-click the highlighted code.
  4. Choose an option from the "Gemini Code Helper" menu.
